NutriList is a grocery list app with smart classification. Built with Swift ,SwiftUI, Supabase & CohereAPI

## Features
- **User Authentication:** Built with Supabase so you can sign in with Email or Apple ID
- **Smart Classification:** Create grocery items with classification powered by Cohere API
- **Stored in Database:** Your items is stored in Supabase Database
- **Minimalist UI Design:** The App is designed with a Minimalist User Interface
